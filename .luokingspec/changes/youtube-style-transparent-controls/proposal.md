# YouTube 风格透明控制面板

## 需求概述

为视频和音频预览页面的全屏模式添加 YouTube 风格的透明控制面板效果。

**需求类型**：Enhancement

**涉及领域**：前端

## 需求背景

当前 `/mushroom/project/preview` 路由的视频和音频预览页面，在全屏模式下，控制面板使用深色背景（`bg-bg-2/90`）和圆角阴影效果。用户希望改为类似 YouTube 的透明效果，让控制面板看起来更轻盈，不遮挡视频内容。

## 功能需求

### 1. 全屏控制面板样式改造

**目标效果**：
- 移除外层控制面板的背景色、圆角和阴影
- 为每个控制组添加独立的半透明圆角背景
- 进度条保持无背景

**涉及的控制组**：

#### 视频预览
1. **缩放控制组**：缩小按钮、百分比显示、放大按钮
2. **播放控制组**：播放/暂停按钮、倍速选择、音量控制
3. **时间显示组**：当前时间 / 总时长
4. **右侧按钮组**：设置按钮、全屏按钮

#### 音频预览
1. **播放控制组**：播放/暂停按钮、倍速选择、音量控制
2. **时间显示组**：当前时间 / 总时长
3. **右侧按钮组**：设置按钮、全屏按钮

### 2. 半透明背景样式

**样式规范**：
- 背景色：`bg-black/50` 或 `bg-white/10`（50% 黑色或 10% 白色半透明）
- 圆角：`rounded-lg`
- 阴影：`shadow-lg`（可选，根据视觉效果决定）
- 毛玻璃效果：`backdrop-blur-sm`

### 3. 布局调整

**调整点**：
- 移除外层 `div` 的 `bg-bg-2`、`rounded-lg`、`shadow-xl` 类名
- 为每个控制组添加独立的包裹 `div`，应用半透明背景样式
- 确保各控制组之间有适当的间距

## 用户指引

**用户期望效果**：
- 类似 YouTube 全屏播放时的控制面板样式
- 控制元素"悬浮"在视频上方，而不是在一个深色面板内
- 每组控制元素有自己的半透明背景，便于识别和点击

## 不在范围内

- 非全屏模式的控制面板样式保持不变
- 控制面板的交互行为（自动隐藏、显示逻辑）不改变
- 其他页面的控制面板样式不受影响
